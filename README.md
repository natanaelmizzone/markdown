## Condições

> if, switc, ternarios

Condições é o *metódo* para **validar** se uma ***informação*** é ~~verdadeira~~, caso seja _verdadeira_ a __condição__ __aplicada__ `if` executa somente o código que está dentro dele, com isso temos condições para que o código execute somente o nescessario.

```csharp
if (valor1 == valor2){
        Messagebox.Show("Os números são iguais");
}
```
```js
if (valor1 == valor2){
        console.log("Os números são iguais");
}
```
```php
if ($valor1 == $valor2){
        echo "Os números são iguais";
}
```
```python
if valor1 == valor2:
        print("Os números são iguais");

```
